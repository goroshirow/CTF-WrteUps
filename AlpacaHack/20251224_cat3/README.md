# cat3

## 概要
`cat`コマンド固定でjailする．

## 解法
色々試すと3文字までしか受け付けない制限があるようです．`cat`は正規表現を受け付けて，ファイルの中身を結合して表示できるみたいなので
```bash
cat *.*
```
を実行すると，
```python
# 🐈️ < gift is in ../flag.txt
import os
os.system(f"cat {input("$ cat ")[:3]}")
```
フラグは`../flag.txt`にあるようですが，どう頑張っても3文字に収められないので，次のコマンドを入力しました．

```bash
cat |sh
```

`cat`は何も入力しないとその後，強制終了されるまで`stdin`を`stdout`に流すという挙動を利用して，パイプラインで`sh`に任意のコマンドを入力できます．あとはフラグを出力するだけ．
