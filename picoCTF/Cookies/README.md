# Cookies

## 解法

サイトに入力フォームがありクッキーの名前かどうかを判定してくれる。

InspectorからCookieの情報を見ると```name:"-1"```という要素がある。

試しに```aaa```と入力すると```name:"-1"```のままであったが、例で書いてある```snickerdoodle```を入力すると```name:"0"```になった。

逆にnameの値を1,2,...と増やしていくと色々なクッキーの名前が画面に表示される。そこでこのサイトをBurp Suiteで開いて、```Proxy```からサイトを開いてCookieをIntoruderに投げる。```name```の部分をプレースホルダーに変えてNumbersのペイロードを1から30に設定して送信した。

それぞれのレスポンスを確認すると```name:"18"```のときにクッキーの名前がフラグになっていることが確認できた。

```
picoCTF{3v3ry1_l0v3s_c00k135_a4dadb49}
```
