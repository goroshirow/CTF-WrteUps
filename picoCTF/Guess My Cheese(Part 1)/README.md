# 解法
解くべき暗号が```YHNRHPKHUCHF```と与えられています。
Affine Cipherというものを使っています。

暗号化$`y = a \cdot x + b \pmod {26}`$
復号化$`x = a^{-1}(y - b) \pmod {26}`$


xは平文アルファベットの番号です。(a->0, b->1, ...)

暗号化はチーズの名前しか受け付けないみたいなので、試しに```Emmental```という単語を入れると```HTTHOKBY```が返ってきました。a=0, e=4であることから
$`1 = a \cdot 0 + b \pmod {26}`$
$`7 = a \cdot 4 + b \pmod {26}`$
と２つの連立方程式がたてられるのでa=21, b=1と分かります。

暗号文を$`x = 5(y - 1) \pmod {26}`$で解くと```LEICESTERFEU```になります。これを送信するとフラグがゲットできます。

```
picoCTF{ChEeSy86ec32f4}
```
