# Guess My Cheese(Part 2)

## 解法
今回の暗号文はハッシュっぽい見た目かつワードリストが渡されているので辞書攻撃でしょう。

確認のため暗号文をCrack Stationに入れてみるも該当なし。

次にpythonで辞書攻撃用のスクリプトを用意してSHA256で試してみましたがマッチしませんでした。ヒントを見ると

```
Remember Squeexy, we enjoy our cheese with exactly 2 nibbles of hexadecimal-character salt!
```

との記載あり。ソルトとは平文に乱数を結合してレインボーテーブル攻撃を防ぐ仕組み。

ソルトはヒントより2バイトと予想されるので平文の最後に0x01から0xffまで結合してハッシュ化するがまたしても該当なし。

更に、ソルトを平文バイト列の途中に入れるというループも追加するが該当なし。

試せることをすべて詰め込んだスクリプトを作成した。

- 平文を[そのまま、大文字、小文字]の３つで試す
- エンコード方式を[utf-8, ascii, latin-1]の３つで試す。
- 平文の全ての位置にsaltを挿入する

この方法で試したところマッチを発見し、暗号化方式は
$`SHA256(lower(m)+salt)`$

であることが分かりました！

```
 picoCTF{cHeEsY5d45bcdd}
```
